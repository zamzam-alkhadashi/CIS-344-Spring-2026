# Online Pet Supply Store Project

This project contains the complete database design and documentation for an online pet supply store. The folder structure is organized to separate the database models, SQL scripts, ER diagrams, and process documentation.

## Folder Structure

### 1. Database and SQL Scripts
- **PetStoreModel.mwb** – The main MySQL Workbench model file containing all tables, relationships, and the UML ER diagram.  
- **PetStoreModel.mwb.bak** – Backup of the Workbench model file.  
- **SQL Create Structure.sql** – SQL script generated from the Workbench model to create all tables, constraints, and relationships in the database.

### 2. ERD
- **chens_notation.jpg** – Chen-style ER diagram drawn by hand showing entities, attributes, and relationships.  
- **uml_from_workbench.png** – UML-style ER diagram exported from MySQL Workbench showing tables and relationships visually.

### 3. ReadME
- **ReadME** – This file explaining the project structure and the contents of each folder/file.

### 4. Selection and Documentation
- **Process Documentation.docx** – Detailed report on system design requirements, decisions, and workflow in a Word document format.  
- **Process Documentation.txt** – Plain text version of the process documentation for easy reference.  
- **Selection and Documentation.txt** – Project
